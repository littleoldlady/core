The base commands...

Give it a try

To take full advantage of your newly installed NuTyX, here are the ESSENTIAL commmands that you will have to use everyday. This article explain the essential commands and are NuTyX specifics. We strongly recommand to know them.

NuTyX does not (yet) have a graphical interface tool for administrative purposes. You will therefore use the terminal as the normal interface. If you are not yet used to it, that's not a big deal; we will explain the commands to type.

Since NuTyX is a Linux operating system, it's important to respect case in commands. ALL the characters in commands must be in lowercase.

Let's start by opening a terminal. Ready? OK, let's go!

In some cases, a command consists of a single word without any options, but in most cases, a command can handle many options. We suggest that you use the command's help option to get an idea of the options available. Here, we will only check the most commonly used ones:

commandname help
or 
commandname --help
or 
commandname -h

the help usually provides a printout of all the available options.

Check if they are any updates available

The command cards sync will check the latest status of the available updates for your system.

The command cards diff will show a list of outdated packages

sudo cards sync
sudo cards diff
synchronizing /var/lib/pkg/saravane/desktop from http://..
/var/lib/pkg/saravane/desktop//MD5SUM
..
Difference between installed packages and the depot of binaries:
Package                      Installed                   Available in the depot of binaries
gcc                          4.9.1                       4.9.2
glibc                        2.20                        2.21

Updating a package

To update a package, the command cards install -u follow by the package name will be used. It's necessary for the update to be available:

sudo cards install -u gcc

Search for a package

To search for a package, the command: 

sudo cards search search_string

will return all the packages which have the search_string in the name or in the description. For example, if you want to know what browsers are available:

sudo cards search browser

should return something like:

chromium           The open-source project behind Google Chrome, an attempt at creating a safer, faster, and more stable browser. Include PepperFlash and libpdf plugins
firefox            Standalone web browser from mozilla.org
...

Add a package

To add a package, the command cards install is used, followed by the name of the package to be installed:

sudo cards install xfce4

The download of the application xfce4 starts immediately. Once the package is downloaded, its dependencies will be downloaded as well. When the download is finished, all the packages will be installed in the right order. In this example, when the command is finished, you will have a working xfce4 desktop installed on your NuTyX.

Add a user

Adding a user under NuTyX is very simple. The command:

sudo nu

will add a user. It can be used with or without options. If used without options, there will be two question to answer:

User name: james
User description: James Bond 007

You should ensure that the user's name doesn't contain any special characters, spaces or capital letters.

If you wish to use the command with options, these must be given in the order user description followed by user name:

sudo nu "James Bond 007" james
Notice the use of quotes to allow the user description to contain spaces.

Alias

It is worth knowing that some commands are actually aliases which are used for simplification. An alias is a shortcut to the original command. To know all the available aliases, just type alias in your terminal.

Conclusion

As you can see the only two commands you need to know are: 

nu
cards

This last command has a more detailed explanation available via its help option

cards help

I wish you a lot of fun with your newly installed NuTyX.
