A Warm NuTyX Welcome To All

I am Thierry Nuttens, the Founder of the NuTyX System. I am Swiss French and bilingual.

NuTyX is a complete GNU/Linux distribution with high performance and flexibility. It is based on the Linux From Scratch books LFS and BLFS.

We recommend that potential users first acquire some knowledge about the GNU/Linux system.

NuTyX is an excellent operating system for persons who want to commit themselves in developing their skills further and learn how a Linux system is put together.

NuTyX is a binary distribution optimized for i686 and x86_64 architectures.

It is up to you to decide whether to use the ports system to compile and install your own source packages or to be content with the binary packages that are natively integrated into the system.

NuTyX offers the GNOME, MATE, LXDE, KDE5, enlightenment and XFCE GUI in binary form.

The 10 benefits of NuTyX

  1. You can install it from almost any other distribution, thanks to the BASH script located at http://downloads.nutyx.org/install-nutyx. This can be downloaded. Study it carefully to see what it will do, then launch it from the host distribution. Briefly, the script will check for all the necessary tools, then start the download of a minimal set of packages and finally, install them in the location you specified. Details and explanations are found in http://www.nutyx.org/en/generate-iso.html.
  2. The hierarchical nature of the ports collection provides maximum flexibility. Each collection is made up of a "mandatory" and an "extra" part.
  3. It allows you to use a very simple method called porting to download a recipe from our repositories. This recipe is very similar to the scripts used in CRUX, and is usually less than 10 lines long. It can also determine and apply all needed Dependencies. Instead of the script, you can also use the binary package manager cards to download and install your packages.
  4. It has the ability, once installed, to be modified easily in order to produce a personalized live version of NuTyX. The new live version can also be re-installed on other machines, should you so desire.
  5. It allows you to install a complete base binary package or a selection of individual base binary packages, for example: devel, man, doc and lib.
  6. It has no need of a separate HDD or partition. The NuTyX installer creates a virtual environment and therefore is able to install/re-install without formatting the target disk or partition.
  7. It has unique installation coding, that prevents the NuTyX installer from FORMATTING the target partition. This means that NuTyX can be installed on a partition containing data: (for example) personal files and various non-system folders.
  8. It has a very powerful, yet simple command which allows you to revert to the base system and completely change the graphical user interface (for example from KDE to XFCE). This makes it unnecessary to reinstall or download every package again. The downloaded packages and updates will be simply be reinstalled automatically.
  9. It has a totally unique package manager made of expandable modules. This has no dependencies except glibc. One of its strong points is the ability to discover binary (run-time) dependencies automatically.
  10. It has the ability to compile all the packages properly per level in a dedicated chroot environment. After compilation, all the dependencies are again checked to make sure no broken links are found.
